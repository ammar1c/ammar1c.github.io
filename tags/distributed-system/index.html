Intro to Distributed System

2023-09-24
#distributed system

This is about distributed systems and how to fail early and often. What Distributed systems: Leslie Lamport: “A distributed system is a system in which a computer that you didn’t know existed can render your computer unusable” Why Distributed systems are used when the capacity of a single computer isn’t enough to meet our needs. A single computer have finite resources such network bandwidth, compute power, memory and persisten storage.
